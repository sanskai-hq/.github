# SanskAI

We build a bespoke AI operating system for each customer — one repository per product, deployed independently, engineered to a shared standard.

- **How we build:** the engineering standard and operating process live in [`sanskai-hq/dev-ops`](https://github.com/sanskai-hq/dev-ops) — start at its `START-HERE.md`.
- **Every repo inherits** quality gates, a type-check gate, and a "loop-ready" structure from [`sanskai-hq/repo-template`](https://github.com/sanskai-hq/repo-template).
- **Shared CI** lives in this repo (`.github`).

Quality is automated, not remembered.
